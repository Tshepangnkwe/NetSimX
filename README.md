**NetSimX** is my comprehensive network simulation project for CMPG 325 Computer Networks course. I'm demonstrating practical implementation of network topologies, protocols, and services using Cisco Packet Tracer. This project covers fundamental networking concepts from basic topologies to integrated hybrid networks with enterprise services.

## 🎯 **Project Highlights**

- **5 Core Topologies**: Bus, Mesh, Star, Ring, Extended Star implementations
- **Hybrid Architecture**: Multi-topology integration with IPv4/IPv6 dual-stack
- **Enterprise Services**: HTTP/DNS/DHCP server configurations with security layers
- **VLAN Segmentation**: Advanced network partitioning and security implementation
- **Complete Documentation**: Detailed IP addressing schemes, configuration notes, and testing protocols

## 🚀 **Quick Start**

### Prerequisites
- Cisco Packet Tracer (Latest Version)
- Basic networking knowledge
- Understanding of IP addressing and subnetting

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/NetSimX.git
   cd NetSimX
   ```

2. **Open topology files**
   - Navigate to `/topologies/` directory
   - Open `.pkt` files in Cisco Packet Tracer
   - Follow configuration guides in respective folders

3. **Review documentation**
   - Check IP addressing tables in `/documentation/ip-addressing-tables/`
   - Follow configuration notes for each topology
   - Reference testing protocols for validation
