# Packet Tracer Files

This folder contains all the .pkt files for my NetSimX project.

## Files to Create During Implementation

- `01-bus-topology.pkt` - Bus topology implementation
- `02-mesh-topology.pkt` - Mesh topology with OSPF routing
- `03-star-topology.pkt` - Star topology with VLANs
- `04-ring-topology.pkt` - Ring topology with STP
- `05-extended-star.pkt` - Extended star hierarchical design
- `06-hybrid-integration.pkt` - Combined hybrid network
- `email-server.pkt` - Email server implementation

## Usage

Each .pkt file corresponds to a section in the main IMPLEMENTATION.md guide. Save your completed topologies here as you work through the project.
