# CMPG 325 Computer Networks Individual Project Requirements

## Project Overview
**Course**: CMPG 325 Computer Networks  
**Institution**: North-West University (NWU)  
**Faculty**: Natural & Agricultural Science  
**Department**: Computer Science  
**Total Marks**: 100  
**Submission Deadline**: 13-October-2025 @23:55  

## Project Objectives

### Part I - Network Topologies Design & Simulation using Cisco Packet Tracer (60%)
**Weight**: 70 marks

#### Required Topologies:
1. **Bus Topology**
2. **Mesh Topology** 
3. **Star Topology**
4. **Ring Topology**
5. **Extended Star Topology**

#### Hybrid Topology Requirements:
- 1 Hybrid topology integrating elements of the above topologies
- Configured with IPv4 & IPv6 dual-stack
- VLAN segmentation implementation
- At least one server (HTTP/DNS/DHCP)
- Basic security implementation

#### Documentation Requirements:
- Full documentation on GitHub (README.md)
- IP addressing tables
- Network screenshots
- Configuration notes

### Part II - Individual Network Feature Configuration using Cisco Packet Tracer (20%)
**Weight**: 30 marks

#### Requirements:
- Set up and configure an email server
- Demonstrate functionality
- Document configuration process

### Part III - 15-30 min Video Demonstration of parts I & II (20%)
**Weight**: Video demonstration showing:
- Overview of all topologies
- Configuration highlights  
- Server functionality testing
- Network connectivity demonstrations
- Security features explanation

## Submission Guidelines

1. **GitHub Repository**: Complete project with all files and documentation
2. **Video Demonstration**: 15-30 minutes covering both parts
3. **Cisco Packet Tracer Files**: All .pkt files for each topology
4. **Documentation**: Comprehensive README.md with setup instructions
5. **Screenshots**: Evidence of working configurations and testing

## Technical Requirements

- **Software**: Cisco Packet Tracer (latest version)
- **Protocols**: IPv4, IPv6, TCP/IP, DHCP, DNS, HTTP, SMTP
- **Security**: VLAN implementation, basic security protocols
- **Documentation**: Markdown format, clear IP addressing schemes
- **Version Control**: Git with regular commits showing development progress
